DOKUMENTASI LHR 2024

Gambaran Umum

Proyek LHR (Lalu Lintas Harian Rata-rata) dirancang untuk menyederhanakan pengumpulan dan analisis data lalu lintas menggunakan kombinasi sistem offline, aplikasi Android, dan kecerdasan buatan. Proyek ini memastikan pelaporan yang akurat dan efisien bagi otoritas terkait.

Fitur

  • Mode offline untuk tim lapangan.
  • Mode online untuk otoritas pemerintah.
  • Aplikasi Android yang dikembangkan dengan Kodular untuk input data manual.
  • Sistem penghitungan kendaraan berbasis kecerdasan buatan.
  • Integrasi dengan Google Drive untuk penyimpanan dokumen.

Dokumentasi Penyimpanan

Semua data proyek, termasuk file dokumen dan laporan, disimpan secara terorganisir di Google Drive untuk mempermudah akses di masa mendatang. Berikut adalah tabel penyimpanan:

No Nama Letak Keterangan
1 Folder Utama LHR_Proyek_2024 Folder utama proyek yang berisi semua sub-folder dan file terkait.
2 Laporan LHR_Proyek_2024/Laporan Berisi laporan lalu lintas harian.
3 Dokumen LHR_Proyek_2024/Dokumen Berisi dokumen administratif dan teknis.
4 Data Mentah LHR_Proyek_2024/Data_Raw Berisi data mentah yang dihasilkan dari aplikasi lapangan.
