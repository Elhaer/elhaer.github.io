DOKUMENTASI LHR 2024

Gambaran Umum

Proyek LHR (Lalu Lintas Harian Rata-rata) dirancang untuk menyederhanakan pengumpulan dan analisis data lalu lintas menggunakan kombinasi sistem offline, aplikasi Android, dan kecerdasan buatan. Proyek ini memastikan pelaporan yang akurat dan efisien bagi otoritas terkait. Semua dokumentasi file disimpan di Google Drive, dan sisanya masih berada di dalam server lama dan server skupi.

Fitur

  • Mode offline untuk tim lapangan.
  • Mode online untuk otoritas pemerintah.
  • Aplikasi Android yang dikembangkan dengan Kodular untuk input data manual.
  • Sistem penghitungan kendaraan berbasis kecerdasan buatan.
  • Integrasi dengan Google Drive untuk penyimpanan dokumen.
