Overview

Project LHR (Lalu Lintas Harian Rata-rata) is designed to simplify traffic data collection and analysis using a combination of offline systems, Android applications, and artificial intelligence. The project ensures accurate and efficient reporting for relevant authorities.

Features

  • Offline mode for field teams.
  • Online mode for government authorities.
  • Android app developed with Kodular for manual data input.
  • AI-based vehicle counting system.
  • Integration with Google Drive for document storage.

Setup Instructions

  1. Download and install XAMPP or WAMP for the local server.
  2. Place project files in the appropriate folder (e.g., htdocs for XAMPP).
  3. Create a database using phpMyAdmin and import the provided SQL file.
  4. Update database configuration in the config.php file.
  5. Start Apache and MySQL services, then access the website via http://localhost.
